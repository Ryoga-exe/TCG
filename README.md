# What Is
Times-Card-Game（仮）は、作者が、プログラム学習の最終目標として自らに開発を課した、デジタルカードゲームです。永遠の未完成。
現在、友人の[azelazel](https://github.com/azelazel)と[a01sa01to](https://github.com/a01sa01to)からいただいたサンプルプログラムを参考に骨組作成中。
